# RegMT — прототип

Прототип сайта по регистрации медицинских изделий.

## Стек

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Локальный запуск

```sh
npm install
npm run dev
```

## Сборка

```sh
npm run build
npm run preview
```

## Деплой

Прод-деплой в Vercel настроен автоматически из ветки `main`.
